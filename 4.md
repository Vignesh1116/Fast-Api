31. What is routing?

Routing connects URL paths with Python functions.

32. What decorator is used for GET requests?

Example concept:

@app.get("/items")

33. What decorator is used for POST requests?

@app.post("/items")

34. What decorator is used for PUT requests?

@app.put("/items/{id}")

35. What decorator is used for DELETE requests?

@app.delete("/items/{id}")

36. What is APIRouter?

APIRouter helps organize large applications by grouping related routes.

37. Why use APIRouter?

It improves:

Code organization

Maintainability

Scalability

38. What is prefix in router?

Prefix adds a common URL path for all routes in a router.

39. What are tags in FastAPI?

Tags help organize endpoints in API documentation.

40. What is route grouping?

Route grouping organizes related endpoints into modules.
