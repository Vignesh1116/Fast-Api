61. FastAPI is built on Starlette

FastAPI uses Starlette internally.

62. FastAPI uses ASGI

FastAPI applications follow the ASGI standard.

63. FastAPI uses Uvicorn server

Applications are run using Uvicorn.

64. FastAPI supports middleware

Middleware processes requests and responses.

65. FastAPI request lifecycle

Request → Validation → Endpoint → Response.

66. FastAPI supports background tasks

Background tasks run operations after sending response.

67. FastAPI supports event handlers

Startup and shutdown events manage application resources.

68. FastAPI supports WebSockets

FastAPI can build real-time applications.

69. FastAPI supports response models

Response models define output structure.

70. FastAPI supports exception handling

Custom exception handlers manage errors.
