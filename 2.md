11. What is ASGI?

FastAPI uses ASGI, which allows asynchronous web applications.

12. What framework is FastAPI built on?

FastAPI is built on Starlette for web features.

13. What library is used for data validation?

FastAPI uses Pydantic for data validation and serialization.

14. What are path operations?

Path operations are functions that handle specific HTTP requests for a URL path.

Example concept:

@app.get("/users")

15. What are HTTP methods?

Common HTTP methods include:

GET → retrieve data

POST → create data

PUT → update data

DELETE → remove data

16. What is request validation?

FastAPI automatically validates incoming request data using Pydantic models.

17. What is response model?

A response model defines the structure of data returned by an API.

18. What is dependency injection?

Dependency injection allows reusable components such as database connections and authentication logic.

19. What are query parameters?

Query parameters are values passed in the URL.

Example:

/users?age=25

20. What are path parameters?

Path parameters are dynamic values inside the URL.

Example:

/users/{user_id}
